# Snappy English Daily Crossword

A standalone React app for daily crossword puzzles featuring English idioms.

## Setup Instructions

1. Create a new repository on GitHub
2. Clone it locally: `git clone YOUR_REPO_URL`
3. Create the file structure shown above
4. Copy each file content
5. Run `npm install`
6. Test locally with `npm start`
7. Push to GitHub
8. Deploy to Vercel (import your GitHub repo)

## API Integration

Update the `fetchCrossword` function in `src/App.js` to point to your API endpoint.
